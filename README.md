Hello this is a project of mine I guess since I can't code GPT-5.2-Codex did. Here is how to use it.
Trainer.py (This will just run CPU for training)
Trainer.py --gpu (This will use your GPU for training if you have one)
Trainer.py --chat (You wil chat with the ai)
Trainer.py --chat --gpu (Uses the GPU for talking)
Trainer.py --link (Also you to add a link for data collection)
It trains the ai with MediaWiki and Stack Exchange as default.
Download setup.py if you want a little easyer setup process it should install all packages needed and install the code
